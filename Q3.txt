1. Git為分散式版本控制系統，用於管理軟體開發過程中的源代碼。它可以跟蹤檔案的修改歷史，允許多人同時協作開發，並支持不同分支的管理，使得軟體開發過程更加有效率、可靠和方便。


2.
List : 有序的集合，可重複，可以包含任意類型元素，可使用append()remove()等方法進行更新或刪除元素。語法符號為[]，例 : list01 = ['apple', 'banana', 'orange']
Tuple : 有序的集合，可重複，可以包含任意類型元素，但元素一旦創建就無法修改。語法符號為()，例 : tup01 = (1,2,3,4)
Set : 無序不可重複的集合，可進行集合運算，例如交集、聯集、差級。語法符號為{}，例 : set01 = {1,2,3,4}
Dictionary : 無序的集合，由健(Key)值(Value)組成，Key的資料型態通常為string或int，而value可以為任何資料型態。
與list一樣可以進行更新或修改刪除等操作，語法符號為{}，健值間用逗號隔開
